# V48 FULL – Margen por servicio

Fecha: 2026-03-21

## Objetivo
Agregar una capa financiera fina para medir:
- margen por patrulla
- diferencia entre facturado y costo
- porcentaje de ganancia por servicio
- servicios rentables vs servicios con pérdida

## Archivos agregados
- margen-servicio/panel.html
- css/v48-margen-servicio.css
- js/core/v48-margen-servicio.js
- data/v48-margen-config.json

## Resultado
Esta versión deja a RED RURAL listo para identificar qué patrullas dejan ganancia real, cuáles apenas cubren costo y cuáles conviene revisar.
