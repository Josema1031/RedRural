# V52 – Integración real de clientes y contratos

Fecha: 2026-03-21

## Objetivo
Conectar por primera vez la capa de negocio con Firestore de forma real para:
- alta de clientes
- alta de contratos
- listado real de clientes/contratos
- cálculo de estado por vencimiento

## Archivos agregados
- `js/core/v52-clientes-contratos.js`
- `css/v52-clientes-contratos.css`
- `admin/clientes-contratos.html`
- `data/v52-clientes-contratos-schema.json`

## Qué cambia
Esta versión deja de ser solo visual y empieza a usar Firestore en el módulo de clientes/contratos.
