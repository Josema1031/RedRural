# REORGANIZACION V12

## Objetivo
Seguir avanzando sin romper nada y, a partir de esta versión, comenzar a crear **archivos nuevos con nombres en español**.

## Qué se agregó
- `js/core/tablero-ejecutivo-ui.js`
- `js/core/resumen-productor.js`
- `js/core/resumen-productor-ui.js`
- `css/resumen-productor.css`

## Criterio
- **No se renombró** nada de lo ya consolidado en V1-V11.
- Desde ahora, los archivos nuevos priorizan nombres en español.
- Los archivos anteriores se mantienen para preservar compatibilidad.

## Qué aporta esta V12
- Un alias en español para el tablero ejecutivo visible.
- Un resumen específico del productor con foco en:
  - solicitudes propias
  - gasto estimado
  - comisión plataforma
  - neto para prestadores
- Base visual para futuras tarjetas de monetización por productor.

## Próximo criterio recomendado
Desde V13 en adelante, los nuevos archivos deberían seguir esta convención:
- `tablero-*`
- `resumen-*`
- `servicios-*`
- `solicitudes-*`
- `metricas-*`

Sin tocar los nombres históricos ya usados por el proyecto.
